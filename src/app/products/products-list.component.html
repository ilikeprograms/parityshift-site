 Porthole
Porthole is a Chrome extension for Digital marketers which makes it easier to manage and change keywords, suitable for use within AdWords.
30 day free trial is offered, so you can try it out!
Main Features
  • Uses Chrome storage, so you can sign in to your Google account and cary on where you left off. Seamlessly.
  • Provides a section where you can add keywords as part of Ad groups
  • Keywords can easily be changed between different types of modifiers e.g Exact match, Broad match modifier etc
  • Keywords be copied/pasted within each AdGroup
  • Clients can be added to group campaigns
  • Campaigns can hold multiple Ad groups
  • Option to export to CSV
